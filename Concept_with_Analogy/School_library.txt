│   📋 SCENARIO: You are the new librarian. You need to organize all books in the school library.    │
│                                                                                                     │
│   STEP 1: Understanding DATA vs INFO (Review)                                                       │
│   ─────────────────────────────────────────────────────────────────────────────────                │
│                                                                                                     │
│   Raw DATA (Unprocessed):                                                                          │
│   ┌─────────────────────────────────────────────────────────────────────────────────────────────┐  │
│   │ "978-0-545-01022-1", "Harry Potter", "J.K. Rowling", "2007", "Fantasy", "A7", "Available"  │  │
│   └─────────────────────────────────────────────────────────────────────────────────────────────┘  │
│                                                                                                     │
│   Processed INFO (Meaningful):                                                                     │
│   ┌─────────────────────────────────────────────────────────────────────────────────────────────┐  │
│   │ "The book 'Harry Potter' by J.K. Rowling is located at shelf A7 and is Available to borrow"│  │
│   └─────────────────────────────────────────────────────────────────────────────────────────────┘  │
│                                                                                                     │
│   STEP 2: Understanding FIELDS (The Categories)                                                    │
│   ─────────────────────────────────────────────────────────────────────────────────                │
│                                                                                                     │
│   Ask students: "What categories do we need to know about each book?"                              │
│                                                                                                     │
│   Their answers become FIELDS:                                                                     │
│                                                                                                     │
│   ┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐             │
│   │   📚 Book ID     │ │   📖 Title       │ │   ✍️ Author      │ │   📅 Year        │             │
│   └──────────────────┘ └──────────────────┘ └──────────────────┘ └──────────────────┘             │
│                                                                                                     │
│   ┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐             │
│   │   🏷️ Genre       │ │   📍 Shelf       │ │   ✅ Status      │ │   👤 Borrower    │             │
│   └──────────────────┘ └──────────────────┘ └──────────────────┘ └──────────────────┘             │
│                                                                                                     │
│   💡 FIELD = A single category/column of information (like "Title" or "Author")                    │
│                                                                                                     │
│   STEP 3: Understanding RECORDS (One Complete Entry)                                               │
│   ─────────────────────────────────────────────────────────────────────────────────                │
│                                                                                                     │
│   Give each student a sticky note to fill one book's information:                                  │
│                                                                                                     │
│   ┌─────────────────────────────────────────────────────────────────────────────────────────────┐  │
│   │                                                                                             │  │
│   │   Book ID: B001     Title: Harry Potter     Author: J.K. Rowling     Year: 2007            │  │
│   │   Genre: Fantasy    Shelf: A7               Status: Available         Borrower: None        │  │
│   │                                                                                             │  │
│   └─────────────────────────────────────────────────────────────────────────────────────────────┘  │
│                                                                                                     │
│   💡 RECORD = One complete row of information about a single item (one book)                       │
│                                                                                                     │
│   STEP 4: Understanding TABLES (Collection of Records)                                             │
│   ─────────────────────────────────────────────────────────────────────────────────                │
│                                                                                                     │
│   Collect all sticky notes and arrange them in a grid on the board:                                │
│                                                                                                     │
│   ┌─────────────────────────────────────────────────────────────────────────────────────────────┐  │
│   │                                    📚 BOOKS TABLE                                           │  │
│   ├─────────┬───────────────────┬─────────────────┬──────┬──────────┬───────┬──────────┬───────┤  │
│   │ Book ID │      Title        │     Author      │ Year │  Genre   │ Shelf │  Status  │Borrower│  │
│   ├─────────┼───────────────────┼─────────────────┼──────┼──────────┼───────┼──────────┼───────┤  │
│   │ B001    │ Harry Potter      │ J.K. Rowling    │ 2007 │ Fantasy  │ A7    │Available│ None   │  │
│   ├─────────┼───────────────────┼─────────────────┼──────┼──────────┼───────┼──────────┼───────┤  │
│   │ B002    │ The Hobbit        │ J.R.R. Tolkien  │ 1937 │ Fantasy  │ B3    │Borrowed  │ Ahmed  │  │
│   ├─────────┼───────────────────┼─────────────────┼──────┼──────────┼───────┼──────────┼───────┤  │

│   └─────────┴───────────────────┴─────────────────┴──────┴──────────┴───────┴──────────┴───────┘  │
│                                                                                                     │
│   💡 TABLE = A collection of related records organized in rows and columns                         │
│                                                                                                     │
